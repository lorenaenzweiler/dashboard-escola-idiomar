Dashboard de Retenção e Análise de Evasão Acadêmica
Este projeto consiste em um dashboard estratégico desenvolvido para a gestão de uma escola de idiomas, focado em monitorar a saúde financeira e acadêmica através da análise de cancelamentos e permanência de alunos.

🧠 Metodologia e Design (Ref. "A Cor dos Dados")
A interface foi construída seguindo os princípios de Cole Nussbaumer Knaflic, focando na clareza para a tomada de decisão:

Cores Opacas e Foco no Dado: Utilização de uma paleta dessaturada (tons de azul e roxo profundo) para garantir que a cor seja um suporte à informação, e não uma distração. Isso reduz a vibração visual e permite uma análise prolongada sem fadiga ocular.

Redução da Carga Cognitiva: O layout é estruturado em blocos lógicos que guiam o usuário dos indicadores macro (Taxa de Evasão) para o detalhamento micro (Cancelamentos por Professor e Nível), minimizando o esforço de processamento da informação.

Atributos Pré-atentivos: Aplicação estratégica de cores para destacar disparidades, como a correlação entre o tempo de permanência e os motivos de saída, permitindo que o gestor identifique padrões de comportamento em segundos.

🔍 Insights de Negócio
O dashboard permite identificar:

Gargalos Acadêmicos: Professores ou níveis com taxas de cancelamento acima da média.

Motivação da Evasão: Análise qualitativa dos motivos de saída cruzada com o tempo de permanência.

LTV (Lifetime Value): Estimativa de permanência média do aluno antes do churn.

🛠️ Tecnologias Utilizadas
Power BI: Modelagem de dados e visualização.

Figma: Design do background para garantir o respiro visual e alinhamento dos elementos.

DAX (Data Analysis Expressions): Cálculos complexos de retenção e variação temporal.
